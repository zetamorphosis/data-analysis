# Streamlit HR Analytics Dashboard

An interactive HR analytics dashboard built using Streamlit and Plotly to explore employee demographic data across organizational structures.

## Project Overview
This project analyzes employee data from an Excel file and provides interactive visualizations to support HR and workforce analysis. 

The dashboard allows users to explore demographic distributions across multiple organizational categories such as department, directorate, job level, and cluster.

## Features
- Gender distribution analysis
- Religion distribution analysis
- Marital status analysis
- Generation classification (based on birth year)
- Education level visualization
- Regional (birthplace) distribution
- Interactive filtering by:
  - Job Title
  - Unit
  - Directorate
  - Department
  - Section
  - Cluster

## Tools & Technologies
- Python
- Pandas
- Streamlit
- Plotly Express

## Key Highlights
- Dynamic filtering and search functionality
- Automated generation classification (Boomers, Gen X, Gen Y, Gen Z, Alpha)
- Interactive pie charts and bar charts
- Clean and user-friendly dashboard layout
